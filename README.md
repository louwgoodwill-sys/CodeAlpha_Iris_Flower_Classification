# Iris Flower Classification

This project was completed as part of the CodeAlpha Data Science Internship.

## Project Overview

The goal of this project is to classify Iris flowers into three species:

- Setosa
- Versicolor
- Virginica

using Machine Learning techniques in Python.

## Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy

## Machine Learning Model

Random Forest Classifier

## Features

- Loads Iris dataset
- Trains ML classification model
- Evaluates accuracy
- Generates classification report
- Predicts flower species

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python iris_classification.py
```
